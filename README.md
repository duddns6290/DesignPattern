# Design Pattern (Java)

객체지향 설계의 핵심 개념을 이해하기 위해  
GoF 디자인 패턴을 직접 구현하며 학습한 저장소입니다.

각 패턴은 단순 구현이 아니라:

- 문제 상황
- 패턴이 해결하는 방식
- 구조적 역할 분리
- 객체 간 협력 관계


---

### Creational
Factory Method

Singleton  

### Structural

Decorator

Composite

### Behavioral
Command  
Iterator  
State  
Template Method
